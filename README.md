# KSM721-Quantitative-Methods-in-Marine-Science
main : Learning journal and reproducible R scripts for KSM721 Quantitative Methods in Marine and Antarctic Science, focusing on statistical analysis, quadrat sampling, and ecological data analysis.

This repository documents my learning process for KSM721 at the University of Tasmania.

The course focuses on statistical methods and quantitative analysis applied to marine and Antarctic science.

Topics covered so far:

- Quadrat sampling and population estimation
- Mean, variance, and standard deviation
- Standard error and uncertainty estimation
- Histograms and probability distributions
- Density estimation

Tools used:

- R
- RStudio
- tidyverse

Purpose:

To build reproducible statistical analysis workflows for marine science applications such as:

- estimating population density
- analyzing ecological sampling data
- understanding variability in natural systems
